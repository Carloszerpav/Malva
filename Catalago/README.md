# Malva Shop

Sitio web minimalista, femenino y profesional para la tienda online **Malva Shop**, enfocado en maquillaje y accesorios originales.

## Requisitos

- [Node.js](https://nodejs.org/) >= 18

## Instalación

```bash
npm install
```

## Ejecución local

```bash
npm start
```

El servidor quedará disponible en `http://localhost:3000`.

## Despliegue en Railway

1. Crea un nuevo proyecto en [Railway](https://railway.app/).
2. Conecta el repositorio de GitHub que contenga este proyecto.
3. Railway detectará `package.json` y usará el comando `npm start` para levantar el servidor Express.
4. Una vez desplegado, Railway generará una URL pública para tu sitio.

## Estructura

- `index.html`: Página principal.
- `styles.css`: Estilos principales.
- `data.js`: Datos de productos.
- `script.js`: Lógica de renderizado y modal.
- `server.js`: Servidor Express para servir el sitio.
